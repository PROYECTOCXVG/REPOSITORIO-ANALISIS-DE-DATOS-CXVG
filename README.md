# 📊 Proyecto de Análisis Exploratorio de Datos

> Proyecto académico desarrollado en el marco de un Bootcamp de Análisis de Datos.

---

## 📝 Descripción

Este proyecto tiene como objetivo aplicar técnicas de **Análisis Exploratorio de Datos (EDA)** para comprender, limpiar y visualizar un conjunto de datos, identificando patrones, tendencias y valores atípicos.

> ⚠️ **Estado:** En desarrollo 🚧

---

## 🎯 Objetivos

- Realizar la carga y exploración inicial del dataset.
- Limpiar y transformar los datos según sea necesario.
- Generar visualizaciones que faciliten la comprensión de los datos.
- Extraer conclusiones y hallazgos relevantes.

## 🚀 ¿Cómo ejecutar el proyecto?

1. Abre el archivo `.ipynb` directamente en [Google Colab](https://colab.research.google.com/).
2. Ejecuta las celdas en orden secuencial.
3. Asegúrate de tener acceso al dataset en la ruta indicada dentro del notebook.
